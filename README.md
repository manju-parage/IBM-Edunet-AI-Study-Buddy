# AI Study Buddy

AI Study Buddy is an intelligent AI-powered assistant designed to help students understand topics faster through explanations, study notes, and quizzes using Generative AI.

---

##  Problem Statement

Students often struggle to understand complex topics quickly and lack personalized learning support.  
AI Study Buddy solves this problem by providing instant AI-generated explanations, notes, and quizzes to improve learning efficiency.

---

## Features

1. Topic explanation using AI  
2. Automatic study notes generation  
3. Quiz generation for self-assessment  
4. Simple and interactive Streamlit interface  
5. Fast AI response system  

---

##  How It Works

1. User enters a study topic
2. Gemini AI processes the request
3. The application generates explanations, notes, and quizzes
4. Results are displayed instantly using a Streamlit interface

---

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- Generative AI
- Prompt Engineering
